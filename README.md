# Annual-sales-Customer-analysis
Project Overview
This project is an interactive Excel dashboard created to analyze annual sales and customer data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Slicers

##  Features
- Orders vs Sales Analysis
- Men vs Women Comparison
- Order Status Analysis (Delivered, Cancelled, Returned)
- Top 5 States Sales
- Age vs Gender Analysis
- Sales Channel Analysis (Amazon, Flipkart, Myntra)
- Interactive Filters (Month, Category, Channel)

## Insights
- Majority of customers are women
- Top sales generated from selected states
- Most orders are successfully delivered

## 🚀 Conclusion
This dashboard helps in understanding sales trends and customer behavior using data visualization.
